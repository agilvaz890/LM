1. ¿Qué es HTML?

HTML (HyperText Markup Language) es un lenguaje de marcas utilizado para crear páginas web. Está formado por etiquetas que indican al navegador cómo mostrar cada elemento (textos, imágenes, enlaces, tablas, etc.).

Estas etiquetas van siempre entre signos menor y mayor (< >). Por ejemplo, <p> se usa para escribir un párrafo.

2. Estructura básica de un documento HTML

<!DOCTYPE html>

Se escribe al principio del archivo, e indica que el documento está escrito en HTML5.

<html>

Encierra todo el contenido del documento.

<head>

Contiene metadatos, enlaces a hojas de estilo, el título y scripts.

  • <meta>: Para definir los metadatos de nuestro sitio web. Los metadatos son información que no se muestra en la página, pero que ayuda a los navegadores, buscadores y redes sociales a entender el contenido del sitio.
    • charset: Define la codificación de caracteres. Lo más común es UTF-8 (permite usar tildes y caracteres especiales).
    • name="viewport": Hace que el sitio sea adaptable (responsive) a diferentes tamaños de pantalla, como móviles y tablets.
    • name="description": Describe brevemente el contenido de la página. Es importante para el SEO (aparece en los resultados de búsqueda).
    • name="keywords": Lista de palabras clave relacionadas con el contenido.
    • name="author": Indica el autor o creador del documento.
    • http-equiv: Define directivas para el navegador (por ejemplo, recargar o redirigir la página).
  • <link>: Para enlazar nuestra hoja de estilos, u otro archivos.
  • <title>: Para escribir el título de nuestro documento. Aparecerá en la pestaña del navegador.

<body>

Contiene todo el contenido visible de la página. A su vez el body, lo podremos dividir en diferentes secciones haciendo uso de unas determinadas etiquetas:

  • <header>: Contiene la cabecera. Dentro del header podemos usar una eqtiqueta <nav> para crear un menú de navegación.
  • <main>: Dentro se incluye el contenido de nuestra página web. Se puede dividir a su vez en diferentes secciones haciendo uso de eqtiquetas como <article>, <section> o <aside>.
  • <footer>: Contiene el pie de página.

3. Etiquetas de texto

<h1> a <h6>

Definen títulos o encabezados, del más grande (<h1>) al más pequeño (<h6>).

<p>

Sirve para escribir párrafos de texto.

<strong> y <em>

<strong> pone el texto en negrita y <em> lo pone en cursiva.

<br>

Inserta un salto de línea.

<hr>

Inserta una línea horizontal para separar contenido.

4. Enlaces

<a>

Se utiliza para crear enlaces o hipervínculos. Permite dirigir al usuario a otra página web, a otra parte o página del mismo documento, descargar un archivo, etc. Sus principales atributos

  • href: Es el atributo principal. Indica la dirección (URL) o destino del enlace, que pueden ser internos o externos.
  • target: Indica dónde se abrirá el enlace.
    • _self: Abre el enlace en la misma pestaña (valor por defecto).
    • _blank: Abre el enlace en una nueva pestaña.
  • title: Muestra un pequeño cuadro de texto al pasar el cursor sobre el enlace.
  • Enlaces internos (anclas): Permiten ir a una parte específica de la misma página.
  • Enlaces con imágenes: Puedes usar una imagen como enlace envolviéndola con <a>.

5. Listas

Las listas en HTML se usan para mostrar elementos relacionados entre sí. Existen 2 tipos principales de listas:

1. Listas no ordenadas <ul>

Como su propio nombre indica (unordered list), se utiliza cuando el orden de los elementos de la lista no es importante. Es lo que se conoce como una lista de viñetas. Su estructura es:


<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
  <li>Elemento 3</li>
</ul>

Ejemplo:

  • Elemento 1
  • Elemento 2
  • Elemento 3

2. Listas ordenadas <ol>

Se utilizan cuando el orden de los elementos sí es importante (ordered list). Los elementos se numeran automáticamente. Se construye usando la siguiente estructura:


<ol>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
  <li>Elemento 3</li>
</ol>

Ejemplo:

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

6. Contenido multimedia

<img>

Se utiliza para mostrar imágenes dentro de nuestra págian web. Es lo que se conoce como una etiqueta vacía, ya que no tiene etiqueta de cierre. Algunos de sus atributos son:

  • src: El más importante. Indica la ruta del archivo de la imagen. Puede ser una dirección local o una URL externa.
  • alt: Se trata de un texto que solo se muestra en el caso de que la imagen no pueda cargarse correctamente.
  • title: Muestra un pequeño texto cuando el usuario pasa el ratón por encima de la imagen.

<video>

Permite reproducir vídeos directamente desde la página web. A diferencia de <img>, esta sí tiene etiqueta de apertura y de cierre. Dentro de ella debe incluirse una etiqueta <source> para especificar la fuente. Sus atributos principales son:

  • src: Es un atributo que se incluye en la etiqueta <source> y especifica la ruta del vídeo.
  • controls: Muestra los botones de control (reproducir, pausa, volumen, etc.).

7. Tablas

<table>

Es la etiqueta que empleamos para crear una tabla. Dentro de ella se utilizan etiquetas específicas para definir sus elementos:

  • <tr>: (table row) crea una fila.
  • <th>: (table header) crea una celda de encabezado.
  • <td>: (table data) crea una celda normal.

Algunos atributos útiles:

  • colspan: Hace que una celda ocupe varias columnas.
  • rowspan: Hace que una celda ocupe varias filas.

Propiedad CSS destacable:

  • border-collapse: Une los bordes que hay entre celdas.

8. Formularios

<form>

Es la etiqueta que utilizamos para indicar que vamos a crear un formualrio. Es su contenedor. Dentro de él introducimos otras etiquetas como:

  • <label>: Texto que describe el campo del formulario. Por ejemplo "Nombre". Principales atributos:
    • for: Indica el id del <input> al que está vinculado.
  • <input>: Crea campos interactivos dentro del formulario. Sirve para que el usuario introduzca datos, seleccione opciones, etc. Sus atributos más importantes son:
    • type: Define el tipo de campo que se mostrará. Sus tipos principales son:
      • text: Campo de texto corto.
      • email: Correo electrónico.
      • number: Campo numérico con flechas.
      • password: Para introducir contraseñas. El texto aparece oculto.
      • radio: Para seleccionar una opción entre varias. Por ejemplo, Hombre/Mujer.
      • checkbox: Para seleccionar una o más opciones.
      • file: Subir archivos.
      • color: Selector de color.
      • submit: Botón que envía el formulario.
      • reset: Botón que resetea los campos.
    • name: Indica el nombre del campo.
    • value: Establece un valor inicial por defecto.
    • placeholder: Texto que aparece dentro del campo y que desaparece cuando se escribe sobre él.
    • required: Hace que el campo sea obligatorio.
  • <select>: Crea una lista desplegable para seleccionar una opción.
    • <option>: Cada elemento dentro de la lista.
  • <textarea>: Crea un área de texto de varias líneas. Por ejemplo, para comentarios. Podemos cambiar el alto con rows y el ancho con cols.
  • <fieldset>: Agruga varios campos relacionados.
  • <legend>: Título de un grupo de campos

9. Canvas

<canvas>

Esta etiqueta se utiliza para crear un área o lienzo donde se pueden dibujar gráficos, animaciones o imágenes mediante JavaScript. Por sí sola, solo define el espacio del lienzo dentro del documento HTML.

  • width y height: Establecen el tamaño del lienzo en píxeles.
